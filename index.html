Codeground.js

A realtime HTML, CSS, and JS plugin for self-hosted demos or playgrounds. Similar to a self-hosted embedded Codepen/JSFiddle.

npm install codegroundjs --save
bower install codegroundjs --save
NPM Bower Github

Features

Tabs view

Show one language at a time

Rows view

Show all the languages in rows ontop of each other

Rows view

Show all the languages in rows ontop of each other

Fullscreen

Takes up the entire container

Preset Values

Add preset string value or file for each language

Disable a language

Only show the language related to your demo

No dependencies

Just javascript and css

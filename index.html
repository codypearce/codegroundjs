Codeground.js

A realtime HTML, CSS, and JS plugin for self-hosted demos or playgrounds. Similar to a self-hosted embedded Codepen/JSFiddle.

npm install codegroundjs --save
bower install codegroundjs --save
NPM Bower Github

Features

Tabs view

Show one language at a time

Rows view

Show all the languages in rows ontop of each other

Fullscreen

Takes up the entire container

Preset Values

Add preset string value or file for each language

Disable a language

Only show the language related to your demo

No dependencies

Just javascript and css

How to use

To create a new playground you need to define an element with the id you want to target, then create a new codeground instance passing in the id of the element you created and pass in the options you want:

var codeground = new Codeground('codeground', opts);

This will create a new codeground in that div with your preset options.

Options

var opts = { html: 'demo.html', // add preset html, css, js by adding a path to the file you want css: 'demo.css', js: 'demo.js', htmlStr: '
foo
', // add preset string values for html, css, or js cssStr: '* { color: black}', // this will override the file you provide jsStr: 'console.log('test')', htmlShow: true, // setting these to false hides that languages editor cssShow: true, // if set false on tabs it hides the button jsShow: true, layout: 'half', // whether editor/output takes up full width or half initialFull: 'editors', // this sets the initial view if you choose layout full style: 'rows', // Toggle between rows and tabs view initialTab: 'html', // if tabs are selected this selects the initial shown tab fullscreen: false, // html, body, and container must be 100% also topbar: true, // this sets a top bar, required for the tabs view title: 'Codeground' // optional title for the tab } var codeground = new Codeground('codeground', opts);
